**EmotionPill** — displays Elysia's current expression (the set from `aivtuber/emotion.py`: happy, playful, shy, sad, surprised, neutral), each with its own accent colour and glyph.

```jsx
<EmotionPill emotion="playful" />
<EmotionPill emotion="sad" lang="zh" />
```

`EmotionPill.emotions` lists the valid keys. Drives the avatar's reaction state in the chat UI.
