**Tabs** — segmented pill or underline navigation between views/sections.

```jsx
<Tabs value={tab} onChange={setTab} items={[
  { key: 'home', label: 'Home', icon: '🏠' },
  { key: 'music', label: 'Music', icon: '🎵' },
]} />
```

`variant="pill"` (segmented glass) or `"underline"`. `dark` for plum surfaces.
