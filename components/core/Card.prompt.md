**Card** — rounded translucent glass surface with a soft pink shadow; the default container for content blocks.

```jsx
<Card tone="glass" hover>…</Card>
<Card tone="dark">…on a plum overlay…</Card>
```

Tones: `glass` (62% white + blur), `solid`, `tint`, `dark` (for night surfaces). Set `hover` for the lift interaction; override `padding` as needed.
